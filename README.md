# 🚚 DispatchPro - Truck Dispatch Website

A modern and responsive Truck Dispatch website developed using HTML, CSS, JavaScript, and Bootstrap 5.

## 📌 Project Overview

DispatchPro is a professional trucking and dispatch service website designed to help trucking businesses manage dispatch operations, route planning, driver support, and fleet management.

The website provides a clean user interface, responsive design, and interactive features for a better user experience.

---

## ✨ Features

- Responsive Design
- Modern User Interface
- Bootstrap 5 Framework
- Dark Mode Toggle
- Animated Statistics Counter
- Truck Tracking Demo Section
- Services Page
- Contact Page with Validation
- WhatsApp Floating Button
- Mobile Friendly Layout
- Smooth Animations and Hover Effects
